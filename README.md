**Project Description**

This project allows users to analyse data of given classes. By using _loops_, _spliting_ and _summarising_ data, users will know exactly the number of students passing the exam as well as their highest and lowest scores. 

**Install**

1. `Open` file
2. Analyse valid and invalid data
3. Find `min`, `max`, `mean`, `medium`
4. `Save` the result

**Result**

You should find the result in the “class1_grades.txt" document, or if you analyse another class file (ex: class2.txt), you should search for classname_grades.txt (ex: class2_grades.txt, class3_grades.txt).
