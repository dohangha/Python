**Project Description**
This project allows users to analyse data of given classes. By using _loops_, _spliting_ and _summarizing_ data, users can know the number of students passing the exam as well as their highest and lowest scores. 

**Install**
1. `Open` file
2. Analyse valid and invalid data
3. Find `min`, `max`, `mean`, `medium`.
4. `Save` the result

**Result**
You should find the result in the “class1_grades.txt" document, or if you analyse other class file (ex: class2.txt), you should seacrh for classname_grades.txt (ex: class2_grades.txt, class3_grades.txt).
